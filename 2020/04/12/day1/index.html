#Kali Linux 扫描技术

1. 信息收集

1.1 发现主机

‘’’
IP地址：（ Internet Protocol Address），是一个32位二进制。
IP分为网络地址和主机地址。A类：（1.0.0.0126.0.0.0），B类（128.1.0.0191.255.0.0），C类（192.0.1.0~223.255.255.0）。
‘’’

1.1.1Netmask/CIDR转换

‘’’
netmask -C 192.168.1.0:192.168.1.255

‘’’

1.1.2网络拓扑绘制

‘’’
路由跟踪。
traceroute ip
‘’’

1.1.3扫码主机

NMAP是一个强大的网络扫描工具，三个基本功能：1.探测主机在线。2.端口扫描。3.推断主机所用的操作系统。

‘’’
1.ping扫描
nmap-sP ip地址/网段
快速获取存活状态和开放的端口
2.ARP主动扫描
netdiscover -r ip地址/网段
‘’’

1.1.4监听主机

监听：不主动的向目标主机发送数据包，仅监听网络中的数据包。因为ARP/DHCP都会进行网络广播。

‘’’
1.ARP监听
netdiscover -p
2.DHCP监听
nmap -script broadcast-dhcp-discover
‘’’

1.2域名分析

域名：是由一串用点分隔的名字，用来表示网络上某一台计算机的电子方位，IP--域名。域名可以获取到：域名所有者信息、子域名、服务器地址。

1.2.1域名基本信息

‘’’
1.whois 工具
whois 域名
2.dmitry一体化信息收集
dmitry -w domain
‘’’

1.2.2子域名信息

‘’’
1.Netcraft子域名查询
https://searchdna.netcraft.com/
2.fierce子域名查询
fierce -dns 域名 -threads 100
‘’’

1.2.3服务器信息

‘’’
1.同IP网站查询
IP反查
http://tool.chinaz.com/Same/
https://dns.aizhan.com/
2.绕过CDN查询真实IP
站长工具ping检测
超级ping
‘’’

1.3端口扫描

每个应用的网络层，想要发送数据

‘’’
1.IP扫描
nmap IP地址
2.查看更多信息
nmap -v IP地址
3.扫描多台
nmap IP地址 IP地址
nmap 192.168.1.1,2,3
4.从文件中读取扫描
nmap -il list.txt
‘’’

1.4操作系统识别

‘’’
1.系统信息 和路由跟踪
nmap -A ip
2.识别操作系统
nmap -O ip
